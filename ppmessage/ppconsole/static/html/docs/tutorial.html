PPMESSAGE使用向导。

拥有PPMESSAGE账号
即可在你的网站和应用内启用PPMESSAGE在线客服。

  • 创建一个PPMESSAGE应用

    登录PPMESSAGE, 然后创建PPMESSAGE应用。

    {{ get_user_fullname() }}, 您好，欢迎您使用PPMESSAGE，在这里创建属于您的PPMESSAGE应用。

    每个账户最多创建12个PPMESSAGE应用，与客服人员属于同一个PPMESSAGE应用的用户，客服人员才能为其提供服务；应用名称最好用您对应网站名字或者应用的名字。可以为特定PPMESSAGE应用邀请客服人员。

  • 选择PPMESSAGE应用

    • {{ app.app_name }}

    登录PPMESSAGE, 这里可以列出您创建的PPMESSAGE应用。

    您还没有创建PPMESSAGE应用，在这里创建属于您的PPMESSAGE应用。

    在这里创建更多PPMESSAGE应用。

  • 添加客服到PPMESSAGE应用

    您还没有登录PPMESSAGE，登录PPMESSAGE, 通过电子邮件邀请PPMESSAGE应用的客服人员。

    {{ get_user_fullname() }}, 您好，欢迎您使用PPMESSAGE，在这里邀请PPMESSAGE应用的客服人员。

    {{ get_user_fullname() }}, 您好，欢迎您使用PPMESSAGE，请在上一步中选择一个PPMESSAGE应用。

    如果被邀请用户的电子邮件地址并没有注册为PPMESSAGE用户，PPMESSAGE系统会发送注册邀请邮件。只有已经是PPMESSAGE的注册用户才能被直接加入您的应用作为客服人员。

  • 选择您的应用类型

    • {{ t.name }}

  • 选择PPMESSAGE应用

    • {{ app.app_name }}

    登录PPMESSAGE, 这里可以列出您创建的PPMESSAGE应用。

    您还没有创建PPMESSAGE应用，在这里创建属于您的PPMESSAGE应用。

    在这里创建更多PPMESSAGE应用。

  • 复制代码

    复制中间代码块到您的 WEB 页面中

    index.html
    <!DOCTYPE html>
<html>
<body>
...
    {{ get_ppcom_web_code() }}
    </body>
</html>

    登录PPMESSAGE，指定您的PPMESSAGE应用，代码中将直接使用相应的应用信息。现在您没有登录，PPMESSAGE应用信息用默认信息替代。

    WEB端SDK代码集成的详细信息请参考WEB SDK。

  • 查看你的网站

    刷新您的网站看看效果。

    如果成功了，那么在您网站页面的右下角就有一个浮动的客服按钮，有了它就可以和您的客服人员即时通讯。

  • 使用WEB端客服软件----开始服务

    登录PPMESSAGE后，可以使用WEB端客服软件开始服务。

    使用WEB端客服软件开始服务。

  • 使用移动端客服软件----开始服务

    使用手机扫描右侧二维码，下载移动端客服软件。
